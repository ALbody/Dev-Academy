اختبار الوحدة 📝

مرحبًا بك في وحدة تطوير الويب 💻

ستتعلم هنا ستة دروس أساسية: HTML، CSS، وJavaScript. استخدم زر "قائمة الدروس" لفتح القائمة واختيار الدرس الذي تريد استكشافه. استمتع بالتعلم التفاعلي!

درس 1: أساسيات HTML 📝

HTML هي لغة ترميز تُستخدم لبناء هيكل صفحات الويب.

درس 2: العناصر والعلامات الأساسية 📝

تتكون صفحات HTML من عناصر تحتوي على علامات (<h1>, <p>, <img>) لتحديد العناوين، الفقرات، والصور.

درس 3: تصميم الصفحات باستخدام CSS 🎨

CSS تُستخدم لتصميم صفحات الويب وتحديد الألوان والخطوط والتخطيطات.

درس 4: التنسيقات النصية والألوان 🎨

يمكن تغيير الخطوط، حجم النصوص، المحاذاة، والألوان باستخدام خصائص CSS مثل font-family, color, وtext-align.

درس 5: أساسيات JavaScript ⚡

JavaScript لغة برمجة تُستخدم لإضافة التفاعلية للصفحات مثل الأزرار والنماذج.

درس 6: المتغيرات والشروط ⚡

المتغيرات تُستخدم لتخزين البيانات، والشروط (if) للتحكم في سير البرنامج بناءً على القيم.
