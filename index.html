A progressive learning path where you complete each lesson and pass assessments before advancing.

Each lesson requires:
1. Learn Content • 2. Code Judge • 3. Pass Quiz

1. Introduction to Flutter

Stage: Content

Introduction to Flutter and its core features:

  • Flutter is an open-source framework by Google
  • Uses Dart as its programming language
  • Provides native app development for iOS and Android
  • High performance and beautiful UIs

2. Widgets Deep Dive

Stage: Content

Everything in Flutter is a Widget:

  • Widgets are the building blocks of Flutter
  • StatelessWidget and StatefulWidget
  • Widget tree and composition
  • Built-in widgets: Text, Button, Container, etc.

3. Layouts System

Stage: Content

Row / Column / Stack - Layout Systems:

  • Row arranges elements horizontally
  • Column arranges elements vertically
  • Stack arranges elements on top of each other
  • MainAxisAlignment and CrossAxisAlignment

4. State Management

Stage: Content

Managing application state:

  • setState is the simplest approach
  • StatefulWidget and lifecycle
  • Provider pattern
  • BLoC pattern

  • Push and Pop operations
  • Route arguments
